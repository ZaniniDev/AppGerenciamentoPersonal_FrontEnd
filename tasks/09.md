## Tarefa
Faça a tela para criar um plano de treino.
Use o design 100% fiel ao zeplin no link a seguir: https://zpl.io/OXoLPYk

## Requisitos tecnicos
O backend está configurado para receber na rota /workout-plans/user/{userId} onde o userID será igual ao userid da sessão.

O json esperado pela api é esse:
{
  "name": "",
  "planGoal": "",
  "workoutStartDate": "",
  "workoutFinishDate": "",
  "workoutDays": [
    {
      "name": "",
      "weekDay": "MONDAY",
      "isRest": false,
      "estimatedDurationInSeconds": 1,
      "coverImageUrl": "",
      "workoutTime": "",
      "exercises": [
        {
          "order": 0,
          "name": "",
          "sets": 1,
          "reps": 1,
          "restTimeInSeconds": 1,
          "obs": "",
          "exerciseLoad": "",
          "exerciseId": ""
        }
      ]
    }
  ]
}

## Regras
- O exerciseId por enquanto não terá funcionalidade, então pode enviar como vazio..
- O usuário poderá criar o treino completo
- Caso o usuário ja tenha plano ativo, ao clicar em salvar ele confirma a desativação do plano anterior
- É possível adicionar multiplos exercicios ao plano de treino
- Só é possível criar UM plano de treino por dia (SEGUNDA,TERÇA,QUARTA,QUINTA,SEXTA,SABADO,DOMINGO)
- Adicione na tela o campo "Objetivo de Plano" referente ao planGoal da api
- O campo "coverImageUrl" pode ser enviado como vazio já que não vai ser possível fazer isso no formulário
- Coloque todos os campos do exercicio no formulario, exceto o campo "exerciseId"
- Garanta que o código não devolva com erros
